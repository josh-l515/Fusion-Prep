# Sistemas Operativos


### ¿Qué es un Sistema Operativo?

- Es el **software principal** que gestiona los recursos del computador.
- Permite la **comunicación entre hardware y software**.
- Proporciona una **interfaz para el usuario** (gráfica o de comandos).

![Diagrama de capas](ruta/a/imagen.png)

---

### Funciones Principales

- **Administrar hardware**: CPU, memoria, dispositivos.
- **Ejecutar programas** y coordinar procesos.
- **Gestionar archivos** y almacenamiento.
- **Seguridad y control de acceso**.
- **Interfaz de usuario** (GUI / CLI).

---

### Tipos de Sistemas Operativos

- **De escritorio**: Windows, macOS, Linux.
- **De servidores**: Linux Server, Windows Server.
- **Móviles**: Android, iOS.
- **Tiempo real / embebidos**: sistemas en autos, routers, IoT.

---

### Evolución y Ejemplos

- 1950s–60s: Mainframes con sistemas básicos.
- 1980s: PC con MS-DOS.
- Hoy: Sistemas multitarea, multiusuario, en computadoras, teléfonos, relojes inteligentes, etc.
